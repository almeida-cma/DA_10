📄 Códigos para Criar

1. ClasseA.java (pacote1)

📄 pacote1/ClasseA.java
package pacote1;

public class ClasseA {
    // 1. PUBLIC - acessível em qualquer lugar
    public String nomePublico = "Publico";

    // 2. PRIVATE - acessível só dentro da classe
    private String nomePrivate = "Private";

    // 3. PROTECTED - acessível no pacote e por subclasses
    protected String nomeProtected = "Protected";

    // 4. DEFAULT (sem modificador) - acessível só no mesmo pacote
    String nomeDefault = "Default";

    // Método para acessar o private (permitido, pois está dentro da classe)
    public void mostrarPrivate() {
        System.out.println("Acessando private DENTRO da classe: " + nomePrivate);
    }

    // Método para testar todos os modificadores DENTRO da classe
    public void testarDentroClasse() {
        System.out.println("=== DENTRO DA CLASSE A ===");
        System.out.println("public: " + nomePublico);
        System.out.println("private: " + nomePrivate);    // ✅ funciona!
        System.out.println("protected: " + nomeProtected);
        System.out.println("default: " + nomeDefault);
    }
}

2. TesteMesmoPacote.java (pacote1)

📄 pacote1/TesteMesmoPacote.java
package pacote1;

public class TesteMesmoPacote {
    public static void main(String[] args) {
        ClasseA obj = new ClasseA();

        System.out.println("=== TESTE NO MESMO PACOTE (pacote1) ===");

        // ✅ public - funciona (qualquer lugar)
        System.out.println("public: " + obj.nomePublico);

        // ✅ protected - funciona (mesmo pacote)
        System.out.println("protected: " + obj.nomeProtected);

        // ✅ default - funciona (mesmo pacote)
        System.out.println("default: " + obj.nomeDefault);

        // ❌ private - NÃO funciona (fora da classe)
        // System.out.println("private: " + obj.nomePrivate); // ERRO!

        // ✅ Acessando private via método público
        obj.mostrarPrivate();

        System.out.println("\n✅ No MESMO pacote: public, protected e default funcionam!");
        System.out.println("❌ private NÃO funciona (só dentro da classe).");
    }
}

3. TesteOutroPacote.java (pacote2)

📄 pacote2/TesteOutroPacote.java
package pacote2;

import pacote1.ClasseA;

public class TesteOutroPacote {
    public static void main(String[] args) {
        ClasseA obj = new ClasseA();

        System.out.println("=== TESTE EM OUTRO PACOTE (pacote2) ===");

        // ✅ public - funciona (qualquer lugar)
        System.out.println("public: " + obj.nomePublico);

        // ❌ protected - NÃO funciona (outro pacote, sem herança)
        // System.out.println("protected: " + obj.nomeProtected); // ERRO!

        // ❌ default - NÃO funciona (outro pacote)
        // System.out.println("default: " + obj.nomeDefault); // ERRO!

        // ❌ private - NÃO funciona (fora da classe)
        // System.out.println("private: " + obj.nomePrivate); // ERRO!

        // ✅ Acessando private via método público (funciona, pois o método é public)
        obj.mostrarPrivate();

        System.out.println("\n✅ Em OUTRO pacote: apenas public funciona!");
        System.out.println("❌ protected, default e private NÃO funcionam.");
    }
}

📊 Resumo dos Modificadores de Acesso

Modificador Mesmo Pacote Outro Pacote Dentro da Classe
public✅ Sim✅ Sim✅ Sim
protected✅ Sim❌ Não✅ Sim
default✅ Sim❌ Não✅ Sim
private❌ Não❌ Não✅ Sim
🎯 RESUMO DIDÁTICO:
  • 🌍 public: "Todo mundo pode ver" → aberto ao mundo
  • 🔒 private: "Só eu posso ver" → apenas dentro da classe
  • 🛡️ protected: "Eu e minha família" → pacote + subclasses
  • 📦 default: "Só meus vizinhos" → apenas no mesmo pacote
